# ⏳ ChronosAI - Smart AI Task Manager

An intelligent **Flutter & Dart-driven** task and time management mobile system. It leverages smart priority algorithms, hands-free voice task generation, built-in Pomodoro focus tools, interactive performance analytics, and gamified streak mechanisms to optimize personal productivity and workflow.

---

## 🎯 Objective

The primary goal of **ChronosAI** is to reduce decision fatigue and revolutionize personal productivity. By combining automated task prioritization, speech recognition, structured focus cycles, and gamified rewards into a seamless mobile interface, ChronosAI helps users build consistency and stay focused without clutter.

## 🚀 Scope of the Project

The current scope of ChronosAI encompasses a fully functional, cross-platform mobile productivity solution designed to optimize daily task management and personal time allocation. The application serves a wide demographic—including students, freelancers, software developers, and working professionals.

### Key Highlights:
* ⚡ **End-to-End Task Lifecycle Management:** Seamless task creation, editing, status tracking, and deletion using an optimized local SQLite backend.
* 🧠 **Algorithmic Task Prioritization:** Real-time dynamic ranking of daily workloads based on multi-variable inputs (urgency, required effort, and user importance rating).
* 🎙️ **Hands-Free Voice Input:** Integrated audio processing pipeline that captures spoken voice prompts and converts them into structured task titles using native mobile engines.
* 🎮 **Behavioral Gamification Framework:** A motivational mechanics engine featuring continuous XP accrual, milestone level-ups, and daily active streaks.
* ⏱️ **Time-Boxing & Deep Work Tools:** Built-in Pomodoro focus sessions paired with analytical performance dashboards to visualize productivity metrics over time.
* 🔒 **On-Device Privacy & Offline Persistence:** Complete functional coverage without relying on active internet connections, keeping user data and voice processing entirely local.

---

## ⚙️ System Requirements

### 🟢 Functional Requirements

* **Smart AI Priority Sorting:** Automatically calculates and ranks tasks based on deadline urgency, task effort, and user-assigned importance.
* **Voice-to-Task Input:** Dictate tasks via microphone to convert spoken audio directly into structured task entries.
* **Pomodoro Focus Timer:** Customizable focus timer (e.g., 25-min work / 5-min break) with notification alerts to keep users in deep work.
* **Gamification & XP System:** Earn +10 XP per completed task, maintain daily activity streaks, and level up to boost motivation.
* **Visual Analytics:** View interactive visual charts to track completed vs. pending tasks over time.
* **Full Task Management (CRUD):** Create, Read, Update, and Delete tasks with instant auto-saving to local storage.

### 🔵 Non-Functional Requirements

* **Privacy & Security:** All user data, speech processing, and local database records remain 100% offline.
* **Performance & Latency:** Fast database queries and minimal response delay during voice recognition.
* **Usability & Design:** Modern, clean, and distraction-free dark mobile interface using Material Design.
* **Data Reliability:** Auto-saves task state to prevent any data loss upon unexpected application shutdown.
